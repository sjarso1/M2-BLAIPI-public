Methods

This page gives a high-level, public summary of the analytical approach. It is intentionally non-sensitive and omits internal implementation detail, data-access instructions, and unreleased code.

Data and Analysis Overview

The project works with publicly accessible Burkitt lymphoma genomic, transcriptomic, and clinical datasets. Analyses are carried out in R and Python using version-controlled, containerized workflows so that results can be reproduced from documented code.

Pathway Activity Analysis

Sample-level metabolic pathway activity is inferred from gene-expression data using Gene Set Variation Analysis (GSVA) against curated pathway gene sets, including KEGG, Reactome, and custom Burkitt-lymphoma-relevant collections. Differences in pathway activity across genetic subgroups are assessed using linear models with multiple-testing correction, and analyses are stratified by EBV status to separate subgroup-associated signals from EBV-associated signals.

Feature Engineering

Literature-derived composite metabolic indices summarize biology linked to the disease's principal oncogenic drivers, including MYC activity, EBV-associated metabolic reprogramming, B-cell-receptor and downstream signaling, and tumor-suppressor-related metabolic disruption. Endemic context indicators are designed to represent malaria-related and geographic factors where appropriate metadata are available, with explicit handling of missing values.

Modeling and Validation Plan

Risk modeling is planned using multivariate survival methods, including regularized Cox regression with survival-forest and gradient-boosting comparators. Model performance is evaluated with nested cross-validation and assessed for discrimination and calibration, benchmarked against a clinical-variable-only baseline. Results described elsewhere on this site as planned analyses should be read as the intended approach rather than completed findings.

Reproducibility Principles

  • Version-controlled code and analysis notebooks.
  • Containerized environments for portable, repeatable analysis.
  • Documented, curated gene sets with literature citations.
  • Open-source release of approved, public-facing materials.
