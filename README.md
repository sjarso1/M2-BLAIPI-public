# M2-BLAIPI Public Website

This repository contains the public-facing website for the M2-BLAIPI project:

**Metabolic Pathway Modeling to Support Risk Stratification in Burkitt Lymphoma**

The site provides a public overview of the project's scientific purpose,
research aims, methods, team, public updates, and collaboration contact information.

The internal research repository is maintained separately and is not publicly accessible.
It contains working code, notebooks, project management materials, and internal documentation.

## Website

The public website is published using GitHub Pages.

## Website Content

Public website files are stored in the `docs/` folder.

## Contribution Rule

Only material that has been reviewed and approved for public release should be added to this repository.
Do not upload restricted data, credentials, internal notes, private links, or unreviewed findings.
